# Website Artikel Sederhana (HTML & CSS)

Project ini merupakan website artikel sederhana yang dibuat menggunakan
HTML dan CSS. Website ini menampilkan halaman beranda, daftar artikel,
halaman detail artikel, serta halaman profil seperti Tentang Kami dan
Hubungi Kami.

## Fitur Website
- Halaman Beranda
- Menu navigasi dengan submenu
- Halaman artikel (detail artikel)
- Sidebar berisi berita dan artikel populer
- Halaman Tentang Kami dan Hubungi Kami
- Footer informasi layanan dan kontak

## Teknologi yang Digunakan
- HTML
- CSS

## Struktur Halaman
index.php
article1.php
article2.php
article3.php
tentang.php
hub.php
style.css
images/


## Cara Menjalankan Website
1. Download atau clone repository ini
2. Letakkan folder project di dalam folder `htdocs` (jika menggunakan XAMPP)
3. Jalankan Apache pada XAMPP
4. Akses website melalui browser:
   http://localhost/nama_folder_project

Website ini juga dapat dibuka langsung melalui browser tanpa server
karena bersifat statis (HTML & CSS).

## Catatan
Website ini dibuat sebagai latihan/pembelajaran pengembangan web statis
menggunakan HTML dan CSS, serta belum menggunakan database.
